NeuroBench Studio

Version 1.0 — EEG Foundation Model Platform

A comprehensive platform for EEG foundation model research, providing tools for exploring 200+ EEG datasets (remote & local), discovering 60+ braindecode models + foundation models, and building visual ML pipelines — all with local dataset support and real-time progress streaming.

Key Features
Multi-Model Comparison

Evaluate MAE, JEPA, Contrastive, EEGNet, Deep4Net, and ShallowNet side-by-side

Representation Geometry

Intrinsic dimensionality, CKA similarity, and manifold structure analysis

Cross-Disease Analysis

ADHD, Epilepsy, Sleep Disorders, Neonatal, and TUH EEG datasets

Visual Pipeline Builder

Drag-and-drop ML pipeline builder with preprocessing, model selection, split strategy, and SSE progress streaming

Preprocessing Pipeline

Temporal filtering, downsampling, bad channel detection, ICA artifact removal, re-referencing, segmentation, and baseline correction

Fallback Model System

Automatic fallback to SimpleEEGNet when braindecode models have compatibility issues, with numba JIT disabled for reliability

Real-Time Progress Tracking

Server-Sent Events (SSE) for streaming pipeline progress with live status updates, metrics, and run history

Supported Models
BD Braindecode Models (60+ models auto-discovered)
Braindecode v1.5.2 is installed with 60+ models auto-discovered from the braindecode.models package via the API. When compatibility issues arise, the system automatically falls back to a built-in SimpleEEGNet fallback model.
FM Foundation Models (pre-trained checkpoints)
Foundation models are pretrained on large-scale EEG data and can be loaded for evaluation or used as feature extractors. Checkpoints are stored per experiment under results/*/checkpoints/.

Pipeline Architecture
Data Loading

Load local datasets from data/raw/ directories or stream 200+ remote datasets via EEGDash API from OpenNeuro — all unified under a single interface

Preprocessing

Sequential pipeline: filtering → downsampling → bad channel detection → re-referencing → ICA → segmentation → baseline correction

Model Training

Choose from 60+ braindecode models + 9 foundation models, trained with configurable epochs, batch size, learning rate, and automatic fallback to SimpleEEGNet

Evaluation & Metrics

Accuracy, balanced accuracy, F1, precision, recall, confusion matrix, AUC-ROC, with per-epoch training curves

Pipeline Execution Details

Pipelines run asynchronously in background threads. Progress is streamed via Server-Sent Events (SSE) and stored in results/pipeline_runs.json. Each run includes full metrics, training history, and model configuration for reproducibility. Supports both local datasets (from data/raw/) and remote datasets (streamed via EEGDash from OpenNeuro).
